ポートフォリオ用の簡易掲示板として、保存による永続化を実装しています。無料枠なので、リロードをしても消えることはありませんが、Render再デプロイ・再起動時には消える可能性があります。

このポートフォリオ制作の意図としては、自分がオクトパストラベラーという作品がとても好きなことと、一般的な掲示板のような不平不満の方が目立つような場所ではなく、コミュニティとして作品愛を語り、より広い見方や同じ作品を経ても異なる感じ方をした人々がノンストレスで滞在できる・したいと思えるような場所を作りたいと考えたからです。
POSTにてサーバーがIDと日付を付与し、保存を行う。
DELETEでは、UUID投稿だけを削除し、ダミー投稿は世界観維持や、具体的に投稿時のお手本を提示することで１投稿目のハードルを下げる目的で削除しない。
最新状態は常にサーバーから取得し、フロントで無理に状態を持たないこと。

挑戦した事柄
・Node.js + Express + server.jsの使用
・REST APIの実装
フロントからfetchで操作できるよう実装

意識した事柄
・Renderを用いて、疑似・簡易的でも第三者に見てもらえる物を作る
世界観を重視し、オーバーに感じるほどの余白や色味・フォントの採用
